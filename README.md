# CSV Explorer Pro

CSV Explorer Pro is a Python-based application for exploring, analyzing, and working with CSV datasets efficiently.

This project provides an easy way to:

- Load CSV files
- View dataset structure
- Explore rows and columns
- Analyze data quickly
- Work with tabular datasets inside a simple Python application

---

# Features

- CSV file upload and reading
- Dataset preview
- Column exploration
- Data inspection
- Lightweight and easy to use
- Python virtual environment support

# Installation

## 1. Clone the Repository

```bash
git clone https://github.com/arpitagupta11/csv-explorer-pro.git
```

## 2. Navigate to Project Folder

```bash
cd csv-explorer-pro
```

## 3. Create Virtual Environment

```bash
python -m venv .venv
```

## 4. Activate Virtual Environment

### Windows

```bash
.venv\Scripts\activate
```

### Linux / macOS

```bash
source .venv/bin/activate
```

## 5. Install Dependencies

```bash
pip install -r requirements.txt
```

---

# Run the Project

If using Python:

```bash
python app.py
```

If using Streamlit:

```bash
streamlit run app.py
```
